# Synpangea Music

Original and adapted music from Synpangea — shared for listening, reflection, remixing, study, and creative reuse.

This repository currently includes audio exports, stem files, and lyrics for two song projects:

- **Breathe**
- **Ulysses**

## About

Synpangea Music is a small public archive of songs connected to the broader Synpangea project: care, continuity, emotional grounding, human creativity, and shared meaning.

These tracks are offered as creative artifacts — part music, part reflection, part open invitation.

Listeners are welcome to download, sit with, share, study, and build from the work in accordance with the repository license and attribution guidance below.
